# powersupplywatch

Util to watch power supply.

## Build and install

```sh
cargo install powersupplywatch
```

## Use

```sh
powersupplywatch [--power-supply=</sys/class/power_supply/AC0>] \
  [--sounds=</usr/share/sounds/freedesktop/stereo>] \
  [--plugin=<power-plug.oga>] \
  [--unplug=<power-unplug.oga>]
```
